# Toastyle
Create a toast message quickly and customize it to what you want.

## Installation
#### repositories
```gradle
maven { url 'https://jitpack.io' }
```

#### dependencies
```gradle
implementation 'com.github.eriffanani:Toastyle:1.0.0'
```
